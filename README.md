## Overview

This project provides a booking system where users can:

- browse available courts
- create bookings
- view bookings in a dashboard/list view

## Pages

1. Home: `/`

3. Book a Court: `/book/`
4. My Bookings: `/my-bookings/`

## Tech Stack

- Python 3.12
- Django 6.0.2
- django-allauth 65.14.3
- HTML5
- CSS3
- Bootstrap 5

## Project Structure

```text
booking_app/                  # Django project config
core/
  forms.py                    # Booking form
  models.py                   # Booking model
  urls.py                     # App routes
  views.py                    # View functions
  templates/core/             # HTML templates
  static/core/styles.css      # Shared CSS
wireframes/                   # SVG and PNG wireframes
docs/
  kanban_user_story_template.md
```

## Setup and Run

```powershell
.\.venv\booking_app\Scripts\python.exe manage.py migrate
.\.venv\booking_app\Scripts\python.exe runserver
```

Open: `http://127.0.0.1:8000/`
